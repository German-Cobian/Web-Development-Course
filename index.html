Main Program

Prework

On the first day, our platform’s prep materials will guide you through the fundamentals of programming with HTML, CSS and JavaScript. Set up your web development environment on your computer, and learn to easily communicate with our expert teacher’s assistants via email or Slack.

Module 1: Responsive design with HTML, CSS and JavaScript

In this module you will learn the foundations of web development: HTML and CSS basics; responive design using Flexbox and Bootstrap; JavaScript; DOM manipulation and version control.

Module 2: The Back-end

Module 2 will send you diving into the back-end to become a Full-Stack developer. Learn how to create a server and discover the intricacies of data modeling. Topics include: NodeJS, ExpressJS, Handlebars, MongoDB and Mongoose.

Module 3: The Front-end

The final module will crack open the Front-end. You will begin to master technologies that are relied upon by the biggest tech companies around the world. Topics include: ReactJS, Single Page Apps (SPA) and Component-based frameworks
